cmd

(cmd this args)

With this function we can query Redis like the following:

=> (cmd querier [:ping])
=> (cmd querier [:get "testkey"])
=> (cmd querier [:set "foo" "bar"])

(Note that the escaped strings are for the docstring, and not what you’d actually type.)
